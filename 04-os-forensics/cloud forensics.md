  Since Cloud forensics is a rapidly growing field in Digital Forensics (DFIR), especially with the widespread adoption of AWS, Azure, and Google Cloud in modern infrastructures. Unlike traditional host forensics, cloud investigations focus on logs, identities, storage activity, network traces, snapshots, and shared-responsibility models.
