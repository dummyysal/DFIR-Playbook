# 🤝 Contributing to DFIR Notes

Thank you for your interest in contributing! This repository is a community-driven resource for learning Digital Forensics & Incident Response.

## 📋 Ways to Contribute

### 🐛 Report Issues
- Found a typo or error?
- Something unclear?
- **Open an issue** describing the problem

### 📝 Improve Content
- Add missing tools or techniques
- Clarify complex explanations
- Add practical examples

### 🧪 Add Labs
- New forensic challenges
- Step-by-step walkthroughs
- Tool usage tutorials

### 🔍 Review
- Proofread existing notes
- Test the tools and commands
- Verify accuracy of technical content

## 📝 Contribution Guidelines

### Writing Style
- Use clear, concise language
- Include code examples where applicable
- Add practical context to theoretical concepts

### Formatting
- Use Markdown syntax consistently
- Keep headings hierarchical (H1 → H2 → H3)
- Use tables for comparisons
- Use code blocks for commands

### Image Guidelines
- Place images in `assets/` subfolders
- Use descriptive filenames
- Include alt text for accessibility

## 🚀 Quick Start

1. **Fork** this repository
2. **Clone** your fork locally
3. **Create** a new branch (`git checkout -b feature/your-feature`)
4. **Make** your changes
5. **Push** to your fork
6. **Open** a Pull Request

## 📬 Contact

- Open an issue for questions
- Use discussions for general topics

*Your contributions help build a better learning resource for the DFIR community!*
